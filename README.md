## Run Locally

**Prerequisites:**  Node.js

attention:
Number 6 is guaranteed to win the first prize, and number 30 is guaranteed to win the second prize.
1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
